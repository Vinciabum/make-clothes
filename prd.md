# Product Requirements Document (PRD)

**제품명:** AI Photo Studio Editor (사내 전문가용) - v4 (Final Popup/Modal UI)
**작성일:** 2026-03-19

## 1. 개요 (Overview)
사내 포토그래퍼 및 리터쳐들의 작업 속도와 눈 피로도를 최소화하고 효율성을 극대화한 사내 전용 AI 증명사진 편집 도구입니다. 복잡한 텍스트 UI를 없애고, **팝업창(모달, 옷장 개념)**을 전격 도입하여 화면 전체를 사진 디테일 검수(Before & After)에 넓게 집중할 수 있도록 디자인을 획기적으로 개선했습니다.

## 2. 핵심 목표 (Goals)
1. **작업 공간(뷰어) 극대화:** 썸네일 그리드가 상시 우측 공간을 차지하지 않고, 클릭 시 화면 정중앙에 시원한 팝업창(모달)으로 뜨게 하여, 평소에는 결과물을 아주 크게 확인할 수 있게 합니다.
2. **시각적 카탈로그 (정사각형 룩북):** 팝업창 내부의 의상과 헤어는 이름 텍스트 없이 1:1 정사각형 썸네일로 표출되어 쇼핑몰 카탈로그를 보듯 매우 직관적인 선택을 돕습니다.
3. **유지보수 자유도 (JSON 동적 관리):** 개발 지식이 전혀 없어도 직원들이 `maintenance_guide.md`에 따라 폴더에 사진을 넣고 메모장만 수정하면 즉각 새 옵션이 추가되는 독립 시스템 구축.

## 3. 기능 및 UI/UX 요구사항 (Features)

### 3.1. 메인 패널 및 팝업(모달) UI 개편
*   **메인 컨트롤 패널 (우측):** 
    *   기존의 긴 썸네일 옵션 리스트를 대신하여, `[남성의상 변경]`, `[여성의상 변경]`, `[남성헤어 변경]`, `[여성헤어 변경]` 4개의 큼직하고 명확한 카테고리 호출 버튼만 깔끔하게 배치합니다.
*   **카탈로그 팝업창 (모달 뷰):**
    *   메인 카테고리 버튼 클릭 시, 화면 정중앙에 주변 배경을 어둡게(Backdrop) 처리하는 동시에 큰 팝업창이 오버레이(Overlay)됩니다.
    *   팝업창 안에는 옵션들이 1:1 정사각형 썸네일 이미지 그리드(Grid)로 시원하게 펼쳐집니다.
    *   썸네일 하단이나 카드 내부에는 어떠한 텍스트 명칭(label)도 노출하지 않습니다. 오직 이미지로만 판단하는 직관성 중심 설계입니다.
    *   작업자가 썸네일을 한 번 클릭하는 즉시 팝업창은 자동으로 닫히며, 뒤편의 큰 뷰어 화면에서 바로 AI 합성 작업(Loading)이 시작됩니다.

### 3.2. 데이터 스키마 및 관리 구조 (JSON 동기화)
*   앱 실행(Mount) 시 `public/presets.json` 데이터를 동적으로 불러와 렌더링에 반영. 코드를 다시 빌드하지 않아도 됩니다.

### 3.3. 프로페셔널 워크플로우 (Pro UX) 지원
*   **작업용 필수 단축키:** `Spacebar`(꾹 누르는 동안 원본 비교 토글), `Ctrl+Z` / `Cmd+Z`(실행 취소), `Ctrl+S` / `Cmd+S`(고화질 즉시 다운로드).
*   **정밀 줌인 뷰어 (돋보기):** 결과 사진의 합성이 잘 되었는지(어깨선이나 잔머리) 확인하기 위해 마우스 커서를 올리면 모니터 전체에 돋보기처럼 확대되어 보여집니다.

## 4. UI/UX 디자인 가이드 요약
*   어두운 작업실 환경에 최적화된 모던 다크 테마(Modern Dark-mode).
*   썸네일 이미지 비율 훼손을 막기 위한 강제 종횡비 제어 (`aspect-square`, `object-cover`).
*   팝업창(모달) 등장 시 부드럽게 커지는 트랜지션 애니메이션과 바깥(Backdrop) 클릭 시 닫히는 UX 표준 제스처 적용.
