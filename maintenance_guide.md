# AI Photo Studio Editor 관리자 매뉴얼 (유지보수 방법)

본 문서는 개발 지식이 없는 사내 직원(관리자)이 향후 새로운 의상이나 헤어스타일 목록을 스스로 추가하고 관리하는 방법을 설명합니다. 이 과정을 통해 별도의 앱 재설치나 코드 수정 없이 앱을 업데이트할 수 있습니다.

## 1. 새로운 썸네일 룩북 이미지 준비하기
새로운 스타일을 앱에 추가하려면 가장 먼저 화면에 보여질 버튼(썸네일) 이미지가 필요합니다.
1. 고객이 보고 고를 수 있도록 가장 예쁘게 나온 의상/헤어 이미지를 준비합니다.
2. 예쁘게 보이도록 **1:1 비율(정사각형)**로 자르는 것을 권장합니다.
3. 이미지 파일의 형식은 `.png` 또는 `.jpg`를 사용합니다.
4. 해당 파일을 컴퓨터의 `내 프로젝트 폴더/public/thumbnails/` 경로 안에 복사해 넣습니다. 
   *(예시 파일명: `new_navy_suit.jpg`)*

## 2. 시스템에 등록하기 (JSON 파일 수정)
이미지를 폴더에 넣었다면, 이제 앱이 해당 이미지를 읽어올 수 있도록 리스트에 등록해 주어야 합니다.
1. `내 프로젝트 폴더/public/presets.json` 파일을 찾아서 우클릭 후 **'메모장(Notepad)'** 또는 텍스트 에디터로 엽니다.
2. 원하는 카테고리에 맞춰, 맨 아래 괄호 안에 다음과 같은 형식의 항목을 하나 추가합니다.

```json
{
  "id": "new_navy_suit_01",
  "category": "men_outfit",
  "thumbnailUrl": "/thumbnails/new_navy_suit.jpg",
  "prompt": "Change the clothing to a high-quality navy blue modern suit, crisp white shirt..."
}
```

* **id:** 다른 것과 겹치지 않는 고유한 영문/숫자 아이디 입력
* **category:** `men_outfit`, `women_outfit`, `men_hair`, `women_hair` 중 하나 입력
* **thumbnailUrl:** 1번 단계에서 추가한 이미지의 이름 적기
* **prompt:** 해당 썸네일을 눌렀을 때 작동할 AI 영어 프롬프트 적기

> **※ 주의사항 (컴마 규칙):** JSON 파일에서는 리스트 사이에 항상 쉼표(`,`)가 있어야 합니다. 바로 윗줄 항목 끝에 커서를 두고 쉼표(`,`)를 꼭 적어준 뒤 새로운 항목을 추가하세요!

## 3. 바로 확인하기
1. 메모장에서 파일을 정상적으로 **저장(`Ctrl + S`)** 합니다.
2. 기존에 열려있던 AI 사진관 웹 브라우저 창으로 돌아가 **새로고침(`F5`)**을 누릅니다. (서버를 껐다 켤 필요가 없습니다.)
3. 방금 추가한 정사각형 썸네일 카드가 카테고리 탭에 정상적으로 나타나는지 확인합니다.
4. 썸네일을 눌러 정상적으로 옷이 바뀌는지 최종 테스트합니다.
